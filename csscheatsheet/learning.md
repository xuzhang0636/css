# CSS Note

## 3 Methods For Adding CSS

- Inline CSS:Directly in the html element(NO!)
- Internal CSS: Using style tags within a single document
- External CSS:Linking an external .css file(Best Practice)

## Colors In CSS

- Color Names
- HTML5 Color names
- Hexadecimal
- RGB

```CSS
/* HTML5 Color names */
body{
    color:red;
    background:coral;
}
/* Hexadecimal */
h1{
    color:#00ff00;
}

/* RGB */
p{
    color:rgb(255,255,255);
}
```
